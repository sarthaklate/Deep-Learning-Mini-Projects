# LSTM - Temperature Prediction

## 📌 Project Description
This project uses Long Short-Term Memory (LSTM) to predict future temperature values based on past data.

## 🎯 Objective
To forecast temperature using time-series data.

## 📊 Dataset
- Custom dataset (temperature_data.csv)
- Features:
  - Day
  - Temperature

## ⚙️ Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib

## 🧠 Model Architecture
- LSTM Layer
- Dense Layer

## ▶️ How to Run
pip install -r ../requirements.txt  
python lstm_model.py
