# ANN - Customer Churn Prediction

## 📌 Project Description
This project uses an Artificial Neural Network (ANN) to predict whether a customer will leave a service.

## 🎯 Objective
To classify customers into:
- Churn (Leave)
- No Churn (Stay)

## 📊 Dataset
- Custom dataset (churn_data.csv)
- Features:
  - Age
  - Balance
  - Active Status

## ⚙️ Technologies Used
- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Matplotlib

## 🧠 Model Architecture
- Input Layer (3 features)
- Hidden Layers (ReLU)
- Output Layer (Sigmoid)

## ▶️ How to Run
pip install -r ../requirements.txt  
python ann_model.py

## 📈 Output
- Accuracy score
- Churn / No Churn prediction
