# Deep Learning Mini Projects (CNN, ANN, LSTM)

## 📌 Description
This repository contains 3 deep learning mini projects:

1. CNN - Handwritten Digit Recognition  
2. ANN - Customer Churn Prediction  
3. LSTM - Temperature Prediction  

## 🛠 Technologies
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib

## ▶️ How to Run
pip install -r requirements.txt

Run each project:
python CNN_Digit_Recognition/cnn_model.py  
python ANN_Customer_Churn/ann_model.py  
python LSTM_Temperature_Prediction/lstm_model.py  

## 📈 Outputs
- CNN: Digit classification  
- ANN: Churn prediction  
- LSTM: Temperature forecasting
